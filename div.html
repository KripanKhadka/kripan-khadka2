Computer

We can type documents, send emails, play games, and browse the Web using a computer. It can also be used to edit spreadsheets, presentations, and even videos, or create them.

A computer is an electronic tool that manipulates data or information. It can store, retrieve, and process information.

All those parts of a computer that are tangible physical objects are covered under the term hardware. The hardware includes circuits, computer chips, graphics cards, sound cards, memory (RAM), motherboards, displays, power supplies, cables, keyboards, printers and "mice" input devices.
